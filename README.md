# NYSE-Stock-Price-Analysis-Project
This project performs a full financial time-series analysis using R. It retrieves historical stock data, processes daily and cumulative returns, evaluates volatility patterns, and visualizes price trends over time. 
Stock-Price-Analysis-Project

The raw dataset is included in the CSV file.

# The Project includes:

- Historical stock price retrieval (adjusted close, open, high, low, volume)

- Daily returns calculation

- Cumulative returns calculation

- Trend visualization using ggplot2

- Volatility and risk evaluation

- Comparison of performance across multiple companies

- Clean, reproducible financial time-series workflow

# Libraries used in R:

dplyr,
readr,
tidyr,
ggplot2
